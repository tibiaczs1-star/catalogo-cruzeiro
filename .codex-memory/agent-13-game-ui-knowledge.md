## Regra-mae

A interface da PubPaid nao pode parecer app moderno colado por cima do jogo.

Ela precisa ser:

- pixelada
- limpa
- legivel
- discreta
- parte do universo visual

## Regras por tipo

### Placas

- silhueta simples
- fundo solido
- borda clara
- texto curto
- nunca competir com o letreiro

### Botoes pixelados

- forma retangular simples
- texto curto
- 3 estados: normal, hover/foco, pressionado
- borda pixel
- nada de sombra HTML generica

### Baloes

- formato limpo
- rabicho simples
- texto curtissimo
- uso raro e rapido

### Tags

- pequenas
- contraste alto
- 1 a 2 palavras
- cor comunica categoria

### HUD

- ocupar pouco espaco
- preferencia por cantos
- escaneavel em 1 segundo
- nunca roubar o centro da cena

### Mensagens

- uma ideia por bloco
- curtas
- aparecem no momento certo
- tutorial minimo

## Tipografia

- fonte pixel ou quase-pixel coerente
- prioridade para legibilidade
- uma fonte principal

## Cor da UI

- base escura
- texto claro
- ciano para interacao
- ambar para objetivo
- magenta so em pontos especiais
- verde para estado ativo/ok
