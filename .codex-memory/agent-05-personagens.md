# Agente 05 - Personagens para Jogo

Status:

- consolidado a partir da primeira rodada
- observacao honesta do agente: sem acesso interno confiavel a playlist nesta sessao

## Foco

- anatomia estilizada
- proporcao
- roupa
- variacao
- identidade visual
- coerencia entre NPCs e protagonista

## Fontes externas uteis levantadas

- PixelSanctuary - silhouette and story
- Saultoons - pixel art character design
- Sandro Maglione - character animations guide
- artigo JGGAG sobre pixel art em character design
- ResearchGate sobre aplicacao de pixel art
- Creative Bloq sobre pixel art moderno

## Regras permanentes

1. personagem bom comeca por silhueta
2. variacao real vem de postura, corpo, roupa e energia
3. animacao nao salva desenho ruim
4. protagonista e NPCs precisam pertencer ao mesmo mundo visual
5. sprite simplificado demais quebra sobre fundo rico

## Erros proibidos

- corpo-retangulo final
- diferenciar NPC so por cor
- mesma cabeca para todos
- mesma pose para todos
- roupa colada como textura
- membros retos sem articulacao
- aprovar sprite sem teste no cenario real
