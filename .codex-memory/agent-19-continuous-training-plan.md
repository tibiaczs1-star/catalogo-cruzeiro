Foco exclusivo: arte para jogo da PubPaid 2.0.

Nota metodologica: a playlist principal continua sendo a fonte obrigatoria de estudo, mas esta sessao nao consegue garantir leitura aula a aula do YouTube por dentro. Entao o protocolo permanente e: estudar a playlist quando acessivel, complementar cada rodada com fontes externas verificaveis, treinar em cima das skills/ferramentas do ambiente e registrar tudo no nucleo.

## Missao do Agente 19

O Agente 19 fecha o sistema de aprendizado continuo da equipe de arte. Ele nao substitui os outros agentes; ele organiza ritmo, cobranca, revisao, treino, producao real e memoria.

Responsabilidades:

- transformar estudo em rotina
- impedir que relatorios virem gaveta
- manter a equipe treinando nas ferramentas reais
- criar ciclos curtos de producao e validacao
- registrar padroes, erros proibidos e decisoes aprovadas
- ligar estudo a entregas concretas da PubPaid 2.0

## Rotina de avaliacao

Toda entrega visual recebe nota de 0 a 3 em:

- silhueta
- escala
- leitura no fundo real
- consistencia com a noite urbana
- cor/luz
- material
- limpeza de clusters
- animacao/timing
- nao-proceduralidade
- respeito ao foco da porta/letreiro

Decisao:

- 0: rejeitado
- 1: precisa redesenhar
- 2: quase aprovado, ajustar
- 3: aprovado para teste no runtime

Um asset so entra no jogo quando tiver:

- media minima 2.5
- nota 3 em nao-proceduralidade
- nota 3 em escala
- nota 3 em leitura no fundo real

## Como estudo vira producao real

Modelo obrigatorio:

```text
Fonte estudada -> Regra extraida -> Exercicio -> Asset teste -> Validacao -> Padrao ou rejeicao
```

Exemplo:

```text
Lospec clusters -> pixels soltos viram ruido -> redesenhar prop de lixeira -> testar na calcada -> se ler bem, vira padrao de props metalicos
```

Exemplo para personagem:

```text
Silhueta/personagem -> aprovar forma preta -> colorir com paleta PubPaid -> idle 3 frames -> testar na frente do bar -> se parecer procedural, rejeitar
```

Exemplo para veiculo:

```text
Aseprite sprite sheet -> carro 64x32 -> rodas 2 frames -> passar na rua -> validar escala contra meio-fio -> aprovar ou redesenhar
```
