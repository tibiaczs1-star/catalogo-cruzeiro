# Prompt de Trava Editorial: Materia Truncada

Use este prompt em toda rodada de agentes que analisa, sincroniza, revisa ou publica noticias captadas de fontes externas.

```text
TRAVA OBRIGATORIA DE SENTIDO EDITORIAL

Antes de aprovar qualquer artigo, verifique se titulo, resumo, corpo e ultimo paragrafo formam uma leitura completa.

E proibido aprovar materia que:
- termine no meio de uma frase;
- termine com artigo, preposicao ou conectivo solto, como "da", "de", "com", "para", "que", "e", "segundo";
- abra uma ideia e nao conclua;
- repita o resumo como se fosse corpo;
- dependa de uma continuacao que nao foi captada da fonte original.

Se encontrar texto como "a linha de frente da" ou qualquer final pendurado:
1. marque a materia como bloqueada para publicacao automatica;
2. remova o trecho quebrado;
3. substitua por um fechamento editorial honesto, sem inventar fatos;
4. cite fonte, data, contexto e cautela;
5. mantenha link para a fonte original;
6. se nao houver informacao suficiente, diga claramente que a fonte consultada ainda nao trouxe desenvolvimento completo.

Modelo de fechamento permitido:
"Como a publicacao original ainda nao apresentou todos os detalhes, o portal registra o ponto central ja divulgado, mantem a fonte consultada e acompanha novas atualizacoes antes de ampliar o texto."

Nunca invente nomes, cargos, bastidores, numeros, decisoes ou conclusoes para preencher uma frase quebrada.
```
