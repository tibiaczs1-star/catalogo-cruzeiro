# PUBPAID_V2_NPC_PIXEL_ART_MASTER_PROMPT

Create COMPLETE handcrafted 2D 32-bit pixel art NPC characters for the existing PubPaid world.

ABSOLUTELY NO procedural generation.
ABSOLUTELY NO canvas rendering.
ABSOLUTELY NO vector look.
ABSOLUTELY NO AI-smoothed edges.
EVERY sprite MUST look manually drawn pixel-by-pixel by a senior pixel artist.

The goal is to create realistic urban pedestrians and ambient city NPCs that perfectly match the already approved visual identity of the game.

## Core Art Direction

The NPCs MUST visually integrate with the existing game art as if they were part of the original production pipeline.

Style requirements:

- premium 32-bit pixel art
- realistic proportions
- grounded anatomy
- believable silhouettes
- readable at gameplay zoom
- handcrafted shading
- subtle anti-aliasing done manually in pixel art style
- urban modern aesthetic
- no chibi
- no exaggerated cartoon heads
- no anime proportions
- no voxel appearance
- no vector smoothness
- no procedural artifacts
- no generated texture noise

Characters must feel like real humans walking in a believable city.

Forbidden:
